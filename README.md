# 豆瓣电影界面
本项目是基于VUE、Json-Server、HTML、CSS、JavaScript和Axios技术栈开发的豆瓣电影界面克隆。通过这个项目，我旨在学习和实践前端技术开发，了解Node.js和VUE技术的应用。
## 1.配置开发环境
首先将我们的环境配置成开发环境，并且安装VUE、Json-Server和Axios：
```javascript
npm init -y
npm i json-server -D
npm i axios -D
npm i vue@2 -D
```
